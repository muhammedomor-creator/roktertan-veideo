## 🔒 ফায়ারস্টোর সিকিউরিটি ও ডাটা গার্ড রুলস (Firestore Rules)
আপনার ফায়ারবেজ ডাটাবেজটিকে সুরক্ষিত রাখতে ফায়ারবেস রুলস ট্যাবে নিচের কোডটি নিশ্চিত করুন:
```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    
    // ভিডিও কালেকশন প্রোটেকশন
    match /saved_portal_videos/{videoId} {
      allow read: if true;
      allow create, update: if request.resource.data.videoId == videoId 
                            && request.resource.data.title is string;
      allow delete: if false;
    }
    
    // ভিজিটর ট্র্যাকিং প্রোটেকশন
    match /analytics/{docId} {
      allow read: if true;
      allow create, update: if docId == 'portal_stats';
      allow delete: if false;
    }
    
  }
}
```
