## 🌟 পরিচিতি ও উদ্দেশ্য
**রক্তের টান (ভিডিও পোর্টাল)** হলো বাংলাদেশের অন্যতম জনপ্রিয় ফ্রী রক্তদান প্ল্যাটফর্ম রক্তের টান (emergencyroktolagbe.vercel.app)-এর একটি বিশেষ সহযোগী ওয়েব অ্যাপ্লিকেশন।
এর মূল উদ্দেশ্য রক্তদাতাদের উৎসাহিত করা এবং সাধারণ মানুষের মাঝে মেডিকেল, স্বাস্থ্য ও সঠিক চিকিৎসা সংক্রান্ত নির্ভরযোগ্য তথ্য পৌঁছে দেওয়া। আমাদের পোর্টালটি সরাসরি ইউটিউব থেকে বাংলাদেশের স্বনামধন্য ও বিশেষজ্ঞ ডাক্তারদের দ্বারা তৈরি করা বিভিন্ন রোগ ও তার প্রতিকার সংক্রান্ত সেরা এবং তথ্যবহুল ভিডিওগুলো একটি আধুনিক, দ্রুতগতির ড্যাশবোর্ডে একত্রিত করে।
## 🚀 কেন এই পোর্টাল? (গুগল এসইও এবং মেডিকেল সমস্যার সমাধান)
আমরা এই পোর্টালটিকে এমনভাবে তৈরি করেছি যা গুগলের **SEO (Search Engine Optimization)** ফ্রেন্ডলি। বাংলাদেশে যখনই কোনো সাধারণ মানুষ গুগলে স্বাস্থ্য বা মেডিকেল সংক্রান্ত সাধারণ কোনো সমস্যা নিয়ে সার্চ করবে, তখনই আমাদের এই পোর্টালটি সার্চ রেজাল্টের প্রথম সারিতে উঠে আসবে।
### 🩺 যেসব মেডিকেল ও মেডিসিন সমস্যার সমাধান এখানে পাওয়া যাবে:
 * **রক্তদান ও সচেতনতা:** রক্ত দেওয়ার উপকারিতা, রক্তদানের সাধারণ নিয়ম এবং কারা রক্ত দিতে পারবেন।
 * **হৃদরোগ (Heart Diseases):** উচ্চ রক্তচাপের প্রতিকার, হার্ট অ্যাটাকের লক্ষণ ও করণীয়।
 * **কিডনি ও লিভারের সমস্যা:** কিডনি ভালো রাখার উপায় এবং লিভার ইনফেকশনের লক্ষণ ও সমাধান।
 * **ডায়াবেটিস নিয়ন্ত্রণ:** সঠিক ডায়েট চার্ট, রক্তে শর্করার মাত্রা নিয়ন্ত্রণ এবং ইনসুলিন ব্যবহারের সঠিক নিয়ম।
 * **মা ও শিশুর স্বাস্থ্য সচেতনতা:** গর্ভকালীন যত্ন এবং শিশুর পুষ্টি নিশ্চিত করার উপায়।
 * **মেডিসিন গাইড ও ডাক্তারের পরামর্শ:** যেকোনো ওষুধের সাধারণ ব্যবহার ও তার পার্শ্বপ্রতিক্রিয়া সম্পর্কে সঠিক ধারণা।
## ⚡ মূল আকর্ষণীয় ফিচারসমূহ (Core Features)
আমাদের এই সিস্টেমে বেশ কিছু যুগান্তকারী ও প্রফেশনাল প্রোগ্রামিং আর্কিটেকচার ব্যবহার করা হয়েছে, যা আপনার ইউটিউব কোটা এবং ডাটাবেজ বিল একেবারে বাঁচিয়ে দেবে:
 1. **ইন-মেমরি সুপার ফাস্ট স্ক্রলিং (In-Memory Random Shuffling):** পেজ লোড হওয়া মাত্রই ডাটাবেজ থেকে সমস্ত ভিডিও মেমরিতে চলে আসে। এর পর ইউজার যখন স্ক্রল করবেন, তখন সম্পূর্ণ র্যান্ডমভাবে পরবর্তী ১০টি ভিডিও দেখানো হবে।
 2. **ডুপ্লিকেটহীন অটো-সিঙ্ক এপিআই (Zero-Cost Unique Sync):** প্রতিবার রিফ্রেশে ব্যাকগ্রাউন্ডে ইউটিউব সার্চ এপিআই রান হয়ে ৫টি নতুন ভিডিও খোঁজা হয়। কিন্তু এটি ডাটাবেজে আগে থেকে আছে কিনা তা চেক করতে কোনো ডাটাবেজ কস্ট বা অতিরিক্ত কুয়েরি খরচ হয় না; সরাসরি ব্রাউজার মেমরিতে ইউনিকনেস চেক করে সম্পূর্ণ ইউনিক ভিডিওগুলোই ফায়ারস্টোরে যুক্ত করা হয়।
 3. **স্মার্ট সিঙ্গেল-প্লেয়ার কন্ট্রোল (Thumbnail-to-Play):** পেজ লোডিংয়ের সময় কোনো ভারী আইফ্রেম লোড হয় না, শুধুমাত্র লাইটওয়েট থাম্বনেইল দেখায়। একটি ভিডিও প্লে হওয়া মাত্রই অন্য চলমান ভিডিও স্বয়ংক্রিয়ভাবে বন্ধ হয়ে পুনরায় থাম্বনেইলে ফেরত যায়। ফলে মোবাইল ব্যবহারকারীদের ইন্টারনেট বেঁচে যায় এবং ব্রাউজার হ্যাং হয় না।
 4. **রিয়েল-টাইম লাইভ ভিজিটর কাউন্টার:** ফায়ারবেজ ট্র্যাকিংয়ের মাধ্যমে সাইটে প্রতি রিফ্রেশ বা ইউনিক ভিজিট সংখ্যা লাইভ গণনা করা হয় এবং তা সরাসরি ইউজারকে স্ক্রিনে দেখানো হয়।
## 📂 ফাইল স্ট্রাকচার (Single-File Architecture)
মোবাইল থেকে অতি সহজে নিয়ন্ত্রণ করার সুবিধার জন্য প্রজেক্টটিকে মাত্র একটি ফাইলে গুটিয়ে আনা হয়েছে:
```bash
roktertan-veideo/
└── index.html      # সম্পূর্ণ ফ্রন্টএন্ড, স্টাইলিং, ফায়ারবেজ কানেক্টিভিটি এবং ইউটিউব ইন্টিগ্রেশন।

```
## 🔧 প্রযুক্তিগত সেটআপ (Tech Stack)

## 🔒 ফায়ারস্টোর সিকিউরিটি রুলস (Firestore Rules)
ডাটাবেজ সুরক্ষিত রাখতে এবং হ্যাকিং প্রতিরোধ করতে ফায়ারবেসে নিচের রুলসটি সেট করা হয়েছে:
```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /saved_portal_videos/{videoId} {
      allow read: if true;
      allow create, update: if request.resource.data.videoId == videoId;
      allow delete: if false;
    }
    match /analytics/{docId} {
      allow read: if true;
      allow create, update: if docId == 'portal_stats';
      allow delete: if false;
    }
  }
}

```
